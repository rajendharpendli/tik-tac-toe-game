# tik-tac-toe-game
A simple and interactive Tic Tac Toe Game built using frontend web technologies like HTML, CSS, and JavaScript. This project allows two players to play the classic Tic Tac Toe game in the browser with a clean user interface and responsive design.  
#Features
Two-player gameplay (X and O)
Responsive and user-friendly interface
Win, draw, and restart functionality
Dynamic game updates using JavaScript
Simple and attractive frontend design.
